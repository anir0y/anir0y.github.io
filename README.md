# [Home](https://anir0y.in)

## Setup

1. Install dependencies:  
   ```bash
   npm install
   ```
2. Run development server:  
   ```bash
   npm run dev
   ```
3. Build for production:  
   ```bash
   npm run build
   ```

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Framer Motion
- ESLint
- PostCSS
