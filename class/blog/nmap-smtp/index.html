Nmap Smtp

Lesson 3

What is SMTP

SMTP stands for Simple Mail Transfer Protocol and is used for email delivery from an email client to an email server.

What is enumeration

There are a few different definitions of enumeration. In network security, this is performed as a discovery process of hosts and devices within a network.

Enumeration tools

Scan

Conduct a scan using tools such as Nmap to determine state of the ports servicing SMTP and any other details that you may ascertain.

Nmap and NSE scripts

The Nmap Scripting Engine (NSE) contains a library of scripts including scripts for SMTP enumeration.

The following are some examples that can be gleaned from use of these scripts.

A couple of things:

  • The output has been edited/sanitised

  • I’ve added a -d (debug) and -v (verbose) for stdout.
